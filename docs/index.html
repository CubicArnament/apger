APGer Engine - Документация

Обзор

APGer - это гибкий Python 3.12+ движок сборки для Arch Linux сред, предназначенный для автоматического создания пакетов в формате APGv2 с использованием GitHub Actions.

APGer автоматизирует процесс сборки пакетов из исходного кода, генерации строгих метаданных JSON, контрольных сумм файлов и создания архивов .apg с автоматическими GitHub Releases.

Структура проекта

  • repodata/: JSON файлы, определяющие пакеты
  • .ci/: Движок сборки и конфигурация
  • .github/workflows/: CI/CD рабочие процессы

Процесс сборки

  1. Парсинг JSON рецептов из repodata/
  2. Загрузка исходного кода
  3. Сборка с использованием указанного шаблона
  4. Создание APG пакета
  5. Подпись пакета с использованием PGP

Документация API

Для подробной документации API смотрите сгенерированные HTML файлы в директории docs/html.

Чтобы сгенерировать документацию, выполните:

make html_docs
